5.2.1 v-bind

v-bind 指令

  • 定义：

    v-bind 指令被用来响应地更新 HTML 属性，其实它是支持一个单一 JavaScript 表达式

  • 语法

    • 完整语法：</span >，解释：v-bind 是指令，: 后面的 class 是参数，classProperty 则在官方文档中被称为“预期值”。
    • 缩写语法：</span >，解释：: 后面的 class 是参数，classProperty 则在官方文档中被称为“预期值”。

  • 示例代码

    <template>
  <div id='goods'>
      <img :src="src" alt="">
  </div>
</template>

<script>
export default {
  name: 'goods',
  data () {
    return {
        src:'123.png'
    }
  }
}
</script>
